# route Ride Group Add Passenger

Route:
MethodPath
POST /user/:user/rideGroup/:groupId/passenger
Add passenger to the specified ride group.
Authentication

This route requires a valid user's token. If authentication fails it will return a 401 http status code.

Body Parameters:
Name Type Description
passengerId string The new ride group passenger user's id.
Header Parameters:
Name Type Description
Authorization string The authentication user's token.
Route Parameters:
Name Type Description
:user string Use 'me' to reference the logged-in user or specify a user's id to act on the behalf of that user.
:groupId string The ride group's id of the affecting ride group.
Response Code:
Type Description
204 The passenger addition to the ride group succeeded.
401 The user authentication has failed.
403 The authenticated user does not have permission.
404 The requested group id for the specified user id does not exist.
422 The request content is unable to be processed due to invalid fields.
500 An internal server error.

# route Ride Group Create

Route:
MethodPath
POST /user/:user/rideGroup
Create a new ride group for the specified user.
Authentication

This route requires a valid user's token. If authentication fails it will return a 401 http status code.

Body Parameters:
Name Type Attributes Description
alias string The ride group's alias.
vehicleId string The ride group's vehicle id.
passengers Array.<string> The ride group's passengers user's id.
meetingPoint.latitude string The ride group's meeting point latitude coordinate.
meetingPoint.longitude string The ride group's meeting point longitude coordinate.
toDestination boolean If the ride group is to/from destination.
avoidTolls boolean If the ride group's route must avoid tolls.
recurrent boolean If the ride group is recurrent
date string optional The ride group's realization date and time (iso 8601 format). Applicable only if recurrent false.
weekDays.sunday boolean optional If the ride group schedules to sundays. Applicable only if recurrent true.
weekDays.monday boolean optional If the ride group schedules to mondays. Applicable only if recurrent true.
weekDays.tuesday boolean optional If the ride group schedules to tuesdays. Applicable only if recurrent true.
weekDays.wednesday boolean optional If the ride group schedules to wednesdays. Applicable only if recurrent true.
weekDays.thursday boolean optional If the ride group schedules to thursdays. Applicable only if recurrent true.
weekDays.friday boolean optional If the ride group schedules to fridays. Applicable only if recurrent true.
weekDays.saturday boolean optional If the ride group schedules to saturdays. Applicable only if recurrent true.
departureTime string optional The ride group's departure time (hh:mm format). Applicable only if recurrent true.
Header Parameters:
Name Type Description
Authorization string The authentication user's token.
Route Parameters:
Name Type Description
:user string Use 'me' to reference the logged-in user or specify a user's id to act on the behalf of that user.
Response Code:
Type Description
201 The ride group creation succeeded.
401 The user authentication has failed.
403 The authenticated user does not have permission.

500 An internal server error.

# route Ride Group Get All

Route:
MethodPath
GET /user/:user/rideGroup
Get all user's ride groups.
Authentication

This route requires a valid user's token. If authentication fails it will return a 401 http status code.

Header Parameters:
Name Type Description
Authorization string The authentication user's token.
Query Parameters:
Name Type Description
mode string Specifies if retrieving user's ride groups as a driver or as a passenger. Possible values: driver, passenger.
Route Parameters:
Name Type Description
:user string Use 'me' to reference the logged-in user or specify a user's id to act on the behalf of that user.
Response:
Name Type Attributes Description
id string The ride group's id.
alias string The ride group's alias
driver.id string The ride group driver's id.
driver.name string The ride group driver's name.
driver.avatar.fileName string optional The ride group driver's avatar file name on the server.
driver.avatar.mimeType string optional The ride group driver's avatar file mime type.
vehicle.id string The ride group vehicle's id.
vehicle.fuelType number The ride group vehicle's fuel type. Possible values: 1 (gasoline), 2 (diesel), 3 (gpl), 4 (electric).
vehicle.consumption number The ride group vehicle's consumption in l/100 km or kWh/100 km..
vehicle.emissions number The ride group vehicle's CO2 emissions in g/km.
vehicle.seats number The ride group vehicle's seats.
vehicle.remainingSeats number The ride group vehicle's remaining seats.
passengers[].id string The ride group passenger's id.
passengers[].name string The ride group passenger's name.
passengers[].avatar.fileName string optional The ride group passenger's avatar file name on the server.
passengers[].avatar.mimeType string optional The ride group passenger's avatar file mime type.
toDestination boolean If the ride group is to or from destination.
isRecurrent boolean If the ride groups is recurrent.
route.summary string The ride group route's summary.
route.duration number The ride group route's duration.
route.distance number The ride group route's distance.
route.hasTolls boolean If the ride group route's has tools.
route.startLocation.address string The ride group route's start location address.
route.startLocation.coordinates.latitude number The ride group route's start location latitude.
route.startLocation.coordinates.longitude number The ride group route's start location longitude.
route.endLocation.address string The ride group route's end location address.
route.endLocation.coordinates.latitude number The ride group route's end location latitude.
route.endLocation.coordinates.longitude number The ride group route's end location longitude.
route.polyline string The ride group route's google maps polyline.
route.boundary.southwest.latitude number The ride group route's southwest boundary latitude.
route.boundary.southwest.longitude number The ride group route's southwest boundary longitude.
route.boundary.northeast.latitude number The ride group route's northeast boundary latitude.
route.boundary.northeast.longitude number The ride group route's northeast boundary longitude.
isExpired boolean optional If the ride group is expired. Applicable only to recurrent ride groups. Visible only when mode is driver.
nextScheduleDate string optional The ride group's next schedule's realization date (iso 8601 format). Ignored if ride group is expired.
Response Code:
Type Description

401 The user authentication has failed.
403 The authenticated user does not have permission.
