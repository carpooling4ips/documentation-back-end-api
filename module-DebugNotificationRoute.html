# route Debug Notify User

Route:
MethodPath
POST /debug/notification
Notify specified user.
Body Parameters:
Name Type Description
userId string The user id to send the debug notification.
notificationKey string The notification key to send.
notificationParams string The specified notification parameters.
notificationData string The specified notification extra data.
Response Code:
Type Description
204 The debug notification sent to the user id succeeded.
422 The request content is unable to be processed due to invalid fields.
500 An internal server error.
