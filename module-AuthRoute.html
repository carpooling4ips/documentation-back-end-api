# route Auth Login

Route:
MethodPath
POST /auth/login
Login the user. If the user does not exist, register instead.
Body Parameters:
Name Type Description
token string The user's Microsoft Graph API token.
Response:
Name Type Description
token string The user's JWT authentication token.
name string The user's name.
title string The user's job title.
Response Code:
Type Description
200 The authentication succeeded.
422 The request content is unable to be processed due to invalid fields.
500 An internal server error. Possible causes: The request to the Microsoft Graph API has been unsuccessful.
