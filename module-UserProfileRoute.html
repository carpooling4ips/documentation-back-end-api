# route Profile Edit Avatar

Route:
MethodPath
PUT /user/:user/profile/avatar
Edit specified user's profile avatar image.
Authentication

This route requires a valid user's token. If authentication fails it will return a 401 http status code.

Body Parameters:
Name Type Description
avatar File The new user's profile avatar image file.
Header Parameters:
Name Type Description
Authorization string The authentication user's token.
Route Parameters:
Name Type Description
:user string The user's id of the editing profile. Use 'me' to reference logged-in user.
Response:
Name Type Description
fileName string The user's avatar file name on the server.
originalName string The user's avatar file original name.
mimeType string The user's avatar file mime type.
size number The user's avatar size in bytes.
Response Code:
Type Description
200 The profile avatar update succeeded.
401 The user authentication has failed.
403 The authenticated user does not have permission.
404 The requested user id does not exist.
422 The request content is unable to be processed due to invalid fields.
500 An internal server error.

# route Profile Get Id

Route:
MethodPath
GET /user/:user/profile
Get specified user's profile data.
Authentication

This route requires a valid user's token. If authentication fails it will return a 401 http status code.

Header Parameters:
Name Type Description
Authorization string The authentication user's token.
Route Parameters:
Name Type Description
:user string The user's id of the retrieving profile. Use 'me' to reference logged-in user.
Response:
Name Type Attributes Description
name string The user's name.
title string The user's job title.
avatar.fileName string optional The user's avatar file name on the server.
avatar.originalName string optional The user's avatar file original name.
avatar.mimeType string optional The user's avatar file mime type.
avatar.size number optional The user's avatar size in bytes.
rating number The user's rating average.
ratingCount number The user's total of received ratings.
askedRides number The user's total asked rides.
givenRides number The user's total given rides.
totalAchievements number The user's total achievements.
comments[].text string The user's received comment message.
comments[].date number The user's received comment timestamp.
isBlocked boolean optional True if the logged-in user blocked the retrieving user.
Response Code:
Type Description
200 The profile retrieval succeeded.
401 The user authentication has failed.
404 The requested user id does not exist.
500 An internal server error.
